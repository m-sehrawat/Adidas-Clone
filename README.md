## [Getting Started]

### Execution..
Want to run in your local Machine? Great!

To run, follow these steps:

- Clone my repository `https://github.com/m-sehrawat/Adidas-Website-Clone`
- Install packages `npm install`
- Run the app `npm start / yarn start'
